# Asado Pro

Calculadora de asado para Río Gallegos y clima patagónico. Estima carne, chorizos, morcillas, achuras, carbón, leña y escote por persona según cantidad de comensales, corte, escenario de cocción, temperatura y viento.

## Funciones

- Ajuste por quincho, chulengo o fuego a la intemperie.
- Factor térmico por viento y frío.
- Desglose opcional por hombres, mujeres y niños.
- Lista de compras con progreso.
- Estimación de gastos y costo por persona.
- Consulta opcional de clima real con `wttr.in`.
- PWA con manifest y service worker para uso offline.

## Desarrollo

```bash
npm install
npm run dev
```

## Validación

```bash
npm run lint
npm run build
```
